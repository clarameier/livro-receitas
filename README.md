# Livro de Receitas :woman_cook:

Olá, seja bem vinda (o) ao meu livro de receitas :wave:

Tenho como objetivo neste livro treinar minhas habilidades com o uso do MarkDown, o Git Bash e o GitHub, aumentando assim cada vez mais o meu repositório. Além disso, posso guardar receitas gostosas para mim e para vocês!

O livro vai ser dividido em duas sub-pastas, sendo uma pasta de [receitas salgadas](https://github.com/clarameier/livro-receitas/tree/master/receitas-salgadas) e outra de [receitas doces](https://github.com/clarameier/livro-receitas/tree/master/receitas-doces). Cliquem nos links e assim vocês terão acesso às receitas MAIS gostosas que poderão experimentar.

Caso se interesse no meu perfil me siga no [GitHub](https://github.com/clarameier), ou no meu [Linkedin](https://www.linkedin.com/in/clarameier/)! Estou disponível para dicas, trocas de ideias e mais contatos.

Muito obrigada e ótima diversão nos códigos e na cozinha pra todos os usuários lendo isto!:heart::woman_cook::keyboard:
